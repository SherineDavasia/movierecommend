# 🎬 Movie Recommendation Engine

A mini data science project that suggests similar movies based on your favorite ones using **cosine similarity** and collaborative filtering.

## 💡 Features
- Built with Python & Streamlit
- Movie similarity via cosine distance
- Interactive UI using dropdown + recommendations
- Real-time movie suggestions

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## 🚀 How to Run
```bash
pip install -r requirements.txt
streamlit run recommend_app.py
